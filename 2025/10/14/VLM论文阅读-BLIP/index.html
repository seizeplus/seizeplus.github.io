BLIP: Bootstrapping Language-Image Pre-training for Unified Vision-Language Understanding and Generation

1.Motivation

  • 为了解决以往VLP (Vision Language Pre-training)模型无法统一理解和生成能力的问题，本文提出了一个MED的混合专家模型，通过Encoder实现理解任务，通过Decoder实现生成任务。

  • 用于训练的数据集是直接从网络上爬下来的，有很多噪声，以往的VLP模型之所以性能上升，是因为数据量提高的提升覆盖掉了噪声的消极影响，但是这是一种次优的解决方案。

2.整体概述

MED架构：BLIP的预训练包括了三个优化目标(ITC、ITM、LM)，同时训练Image enoder、Text encoder、以及注入图像信息的文本encoder和decoder。MED可以分别作为单模态编码器（包括图像编码器和文本编码器）、图像引导文本编码器和图像引导文本解码器来进行操作，从而完成图文多模态的理解和生成任务。

  • 相对于Text encoder，Image-grounded Text encoder多了一层Cross Attn来嵌入图像特征以对齐文本内容。
  • 对于Image-grounded Text decoder，论文将Bi Self-Att替换为Causal Self-Att，用来生成符合图像和文本特征的文本描述。
  • ITC任务: ITC 用于训练 lmage Encoder 和 Text Encoder，通过对比学习对齐图像和文本的特征空间。具体方法是最大化正样本图像 - 文本对的相似度且最小化负样本图像 - 文本对的相似度，从映射的角度讲就是将语义相关的图像 - 文本对映射到空间中的相邻点且语义不相关的图像 - 文本对映射到不相邻的点。
  • ITM任务: ITM 用于训练 Image-grounded text encoder，通过学习图像 - 文本对的联合表征实现视觉和语言之间的细粒度对齐。具体方法是通过一个二分类任务，预测图像文本对是正样本还是负样本。
  • LM任务: LM 用于训练 Image-grounded text decoder，实现生成图像的文本描述任务。具体方法是通过优化交叉熵损失函数，训练模型以自回归的方式最大化文本的概率。

CapFilt机制

  • 字幕器 Captioner：一个视觉文本解码器，基于Image-grounded text decoder用于生成给定图像的字幕，使用 LM 损失函数在 COCO 数据集上进行微调。给定网络图片I_w，生成T_w。
  • 过滤器 Filter：一个视觉文本编码器，基于 Image-grounded text encoder，用于去除文本噪声，使用 ITC 和 ITM 损失函数在 COCO 数据集上进行微调。Filter 通过比对文本和图像的匹配情况，删除原始 Web 文本T_w和合成文本T_s中的噪声。
  • 具体来说，论文先用噪声数据预训练一版MED，再用经过人工高质量数据微调的MED里面的decoder和encoder来生成更高质量的数据，再用高质量的数据重新训练MED，如此循环。
